Een boodschap blijft zich eindeloos herhalen op het stoffige LED-scherm "Syntax error, please try again". Ze probeert de fout op te sporen maar de code is een warboel... Als ze een credit kreeg voor elke fout was ze nu millionair. "Do Program() while solution==false" "Syntax error..." Waarom konden deze oude machines niet gewoon mensen verstaan? Voorzichtig bladerde ze door de oude tekst, het papier bruin van de jaren en koffie vlekken, naar pagina 420. "Hoofdstuk 4: 'For', 'While' en 'Do while' " "Dit gaat een lange nacht worden," denkt ze terwijl ze nieuwe code ingeeft via keyboard, " gelukkig heb ik alle tijd van de wereld." Haar schip drijft steeds verder weg van de zon, richting haar nieuwe thuis.
